# Orchestration

## Work packets

- `R1-layout`: 在 320/390/768/1024/1440px 记录关键区域的尺寸、溢出、按钮换行与可达性。只读。
- `R1-states`: 审查默认、选中、未选中、禁用和焦点状态的色彩与可访问性。只读。
- `R2-main`: 根据基线实施最小、可验证的修正。
- `R3-review`: 在相同断点复审实际页面与交互。
- `R4-fix`: 仅处理 R3 阻塞项，然后执行测试、类型检查、构建与最终浏览器验证。

## 集成规则

审查代理不修改代码。主代理只接受可由真实页面或源码证据支持的问题。任何修正不重置已有工作树改动。
