# Change: 支持多图输入预览与自动下载开关

## Why
- 现有流程假设单张图片，无法高效处理多图批量裁切与下载，用户需重复上传与点击。
- 自动下载缺乏显式的开关与持久化，易被浏览器拦截或产生误触发。

## What Changes
- 支持文件选择与拖拽同时输入多张图片，形成可遍历的待处理队列。
- 预览区域同步展示多张图片及其裁切预览，保持顺序与可见性。
- 批量下载流程支持手动触发与按队列顺序执行。
- 新增“自动下载”复选框，默认关闭，状态持久化；开启后切片完成即触发下载，关闭时需手动点击下载。

## Impact
- Affected specs: image-batch
- Affected code: 上传/拖拽入口、预览与切片队列管理、下载触发与命名规则、本地偏好存储与 UI 控件
