# Change: 版本信息与移动端网格体验优化

## Why
现有界面缺少版本可见性与品牌入口，用户无法辨识版本并访问仓库；移动端空间紧张，网格信息与提示占据视图，且切换网格时会误触发下载流程，缺少快速清空当前图片的手段。

## What Changes
- 新增头部版本号展示与品牌/logo，右上角提供 GitHub 跳转入口。
- 提供清除按钮，一键移除已加载图片并将应用重置为等待上传状态。
- 调整网格切换行为，在已有图片时仅更新预览，不自动触发下载。
- 优化移动端布局：隐藏网格信息卡/提示、网格预设折叠为两行并可展开。

## Impact
- Affected specs: app-shell, grid-controls
- Affected code: Header/AppShell、网格选择与下载触发逻辑、移动端样式与提示文案
