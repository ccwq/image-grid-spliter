# Change: 切片预览与缩放下载

## Why
- 当前裁切流程仅提供结果列表与批量下载，缺少单张切片的预览体验，不利于快速核对与挑选。
- 用户希望在预览态即可左右滑动查看全部切片（含新生成的），并对单张切片进行缩放与下载，减少往返切换成本。

## What Changes
- 新增切片预览视图，支持左右滑动/点击切换，动态包含已存在和预览期间新生成的切片。
- 在预览态提供缩放（捏合/滚轮/按钮）能力，便于细节检查与移动端查看。
- 预览视图内提供单张切片的下载入口，沿用现有命名规则与浏览器下载能力。
- 允许引入第三方预览库（优先 MIT/轻量、内置或可扩展下载按钮、支持触摸滑动与缩放）以加速开发，先调研再选型。

## Impact
- Affected specs: slice-preview
- Affected code: 切片状态管理、预览组件与手势处理、下载触发与命名规则复用、移动端交互适配、第三方预览库引入与样式覆盖
