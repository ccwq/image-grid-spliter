# 术语表

## 批量导出

将一次裁切生成的多个切片文件保存到用户设备的过程。

## 目录直写

网页取得用户明确选择的目录句柄后，通过 File System Access API 创建并写入文件。不经过浏览器的传统下载队列。

## 目录句柄（`FileSystemDirectoryHandle`）

用户授权给网页的特定目录引用。它可以被持久化以供后续尝试复用，但读写权限并不保证永久有效。

## 传统逐张下载

为每个切片创建并点击 `<a download>` 元素的现有实现。文件位置与“是否每次询问保存”完全由浏览器设置控制。

## 降级路径

目录直写不可用时使用的兼容行为。本功能的降级路径是传统逐张下载，明确不包含 ZIP 打包导出。

## 重名后缀

防止覆盖已有导出文件的命名规则：在扩展名前附加递增的连字符序号，例如 `photo-01.png`、`photo-01-1.png`、`photo-01-2.png`。
